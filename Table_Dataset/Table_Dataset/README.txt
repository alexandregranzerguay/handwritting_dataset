
This dataset was used in experimental work in following paper:

Jayant Kumar, Peng Ye and David Doermann. "Learning Document Structure for Retrieval and Classification." International Conference on Pattern Recognition (ICPR 2012), 2012. 

Please cite the paper if you are using this dataset.

RELEVANT: This folder contains tables images (relevant images)
A total of 216 images.
NOTRELEVANT: This folder contains image without tables (irrelevant images)
A total of 402 images.

In our experiments, multiple iterations of randomly chosen 400 training images and 132 test images were perfoemed. We used 66 rel. and 66 irrelevant images for test. 
To create imbalance 150 rel. and 250 irrel. images were selected.
